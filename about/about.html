What's New!!

This new version of FElion Spectrum Analyser has major update to its GUI interface and the data visulaisation methods.

  • The GUI layout is written in HTML 5 and styled using CSS 3
  • The functionality is controlled by Javascript
  • All the data analysis methods are computed by Python
  • This software by itself exist through ElectronJS library runs via NodeJS runtime environment (using Google Chrome's V8 engine)

  • The plots are rendered using plotly.js which is a JavaScript Open Source Graphing Library
  • The logical computations are written using various Python libraries>
